# Minimal Casino Project

This workspace contains a minimal, educational online-casino prototype.

Structure:
- `server/` - Express backend serving a tiny static frontend and simple game APIs (in-memory data).

Quick start (macOS / zsh):

```bash
cd server
npm install
npm start
# then open http://localhost:3000
```

Notes:
- This project is for learning/demo only. It is NOT secure, does not persist data, and must NOT be used in production.
